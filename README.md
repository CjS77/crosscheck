## Getting Started

Install the module with: `npm install crosscheck`

```js
var crosscheck = require('crosscheck');
crosscheck.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g crosscheck
$ crosscheck --help
$ crosscheck --version
```
